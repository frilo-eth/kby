# Prototypes

This directory contains interactive prototypes integrated into the component collection.

## Structure

```
prototypes/
├── feed/          # Feed V3 prototype (integrated)
└── README.md      # This file
```

## Feed V3

The Feed V3 prototype should be copied into `prototypes/feed/` directory.

To run Feed V3:
```bash
cd prototypes/feed
npm install
npm run dev
```

The prototype will be accessible via iframe at `http://localhost:5173` when viewing `/components/feed-v3` in the main app.
